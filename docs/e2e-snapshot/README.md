# E2E Snapshot (외부 배포용)

이 디렉터리는 [mcmp-e2e](https://github.com/m-cmp/mcmp-e2e) 저장소의 `deploy/` 폴더 **읽기 전용 스냅샷**입니다.

- **수정 위치**: `mcmp-e2e/deploy/` (canonical source)
- **동기화**: mcmp-e2e 저장소의 `scripts/sync-to-mc-web-console.sh` 실행
- **버전 정보**: [MANIFEST.json](MANIFEST.json)

외부 배포 tarball·릴리스 번들에 E2E 테스트를 포함할 때 사용합니다.
로컬 개발·CI 실행은 mcmp-e2e 저장소를 직접 사용하세요.
